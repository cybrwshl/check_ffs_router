# check_ffs_router

## Usage
````bash
usage: check_ffs_router.py [-h] [-u URL] [-n NAME] [-w RANGE] [-c RANGE]

optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     where to get status json from
  -n NAME, --name NAME  router to check
  -w RANGE, --warning RANGE
                        return warning if clients is outside RANGE
  -c RANGE, --critical RANGE
                        return critical if clients load is outside RANGE

```
